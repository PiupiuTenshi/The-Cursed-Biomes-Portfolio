# Local Fonts

Place the DeArPix font file here so every page can use it:

- `DeArPix.woff2` preferred
- `DeArPix.woff`, `DeArPix.otf`, or `DeArPix.ttf` fallback

Current bundled file:

- `DeArPix.ttf` copied from the installed `DeArPix 1.94 Regular` font.

The site also checks locally installed fonts named `DeArPix`, `DeArPix 1.94 Regular`, `DearPix`, or `DeArPix Viet Hoa`.
