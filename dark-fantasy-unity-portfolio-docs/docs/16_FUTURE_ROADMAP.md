## 2. Advanced ideas

### AI chatbot over your repo/docs

Bot có thể trả lời dựa trên:

```txt
README.md
project case studies
CV
skills
repo metadata
```

Flow:

```txt
User asks -> retrieve docs -> answer -> suggest project/CV/contact
```

### Project case study pages

Mỗi featured repo có route:

```txt
/projects/unity2dtopdown
/projects/pii-shield
/projects/botai
```

Case study gồm:

- Problem.
- Role.
- Tech stack.
- Architecture.
- Screenshots.
- Challenges.
- What I learned.
- Future improvements.

### Recruiter analytics

Dashboard có thể hiển thị:

```txt
Which project recruiters clicked most
Which section gets viewed longest
Which language visitors prefer
Which CTA converts best
```
